page.logicAPI();

GET /logic

Parses and evaluates a logical expression whether in pseudocode, code, or logic.
NOTE: When typing in the URL manually, you'll need to encode the & symbols and the | symbols.
Before calling the API, parse the expression so it is URL-safe.
The download and raw parameters can only be used by someone in the browser.

Parameters

expression

string The logical expression to parse and evaluate.

table

boolean Whether or not to receive a truth table for the logical expression. Defaults to false.

download

boolean Whether or not to download the file directly to your device.

raw

boolean Whether or not to show the raw truth table in the browser.

Response 200

Body

{
  "success": true,
  "value": [
    {
      "expression": "a",
      "truth_value": {
        "a": true
      },
      "value": true
    },
    "... More Expressions ..."
  ]
}

Response 200

Body

{
  "success": true,
  "value": [
    "| a | b | a ^ b |",
    "+---+---+-------+",
    "| T | T | T |",
    "| T | F | F |",
    "| F | T | F |",
    "| F | F | F |"
  ]
}

Response 400

Body

{
  "success": false,
  "error": "You need an expression to parse."
}

Response 400

Body

{
  "success": false,
  "error": "You need a truth value for the variable "a""
}

Response 400

Body

{
  "success": false,
  "error": "An unknown error has occurred and has been sent to the developer."
}

page.llamaAPI();

GET /llamas

Retrieves a random line of dialogue from Llamas With Hats.

Parameters

episode

integer The episode number to get a line from. Can be between 1-12. Chooses random episode by default.

author

string The speaker to get text from. Can be Carl or Paul. Chooses random author by default.

any

boolean Whether or not to choose any type of quote. Even one-worded quotes or action lines. Defaults to false.

fullScript

boolean Whether or not to get the full script of quotes. To use this, you must provide an episode to get the full script from. Defaults to false.

Response 200

Body

{
  "success": true,
  "episode": 1,
  "image": "https://i.quotev.com/img/q/u/16/5/30/e37ac6e9b1-supe.jpg",
  "author": "Carl",
  "value": "My stomach was making the rumblies."
}

Response 200

Body

{
  "success": true,
  "episode": 1,
  "image": "https://i.quotev.com/img/q/u/16/5/30/e37ac6e9b1-supe.jpg",
  "quotes": [
    {
      "author": "Paul",
      "value": "CAAAARL! THERE IS A DEAD HUMAN IN OUR HOUSE!!"
    },
    "... More Quotes ..."
  ]
}

Response 400

Body

{
  "success": false,
  "error": "No Episode Given."
}

Response 400

Body

{
  "success": false,
  "error": "Invalid Author."
}

Response 400

Body

{
  "success": false,
  "error": "Invalid Episode."
}

page.officeAPI();

GET /office

Gets a random line of dialogue from The Office.

Parameters

season

integer Must be a season in the range 1-9.

episode

integer The number of the episode to get. If the episode number exceeds the amount of episodes in the given season, an error will be given.

type

string The type of quote to get. Can be either aired, deleted, or any. Defaults to aired.

Response 200

Body

{
  "success": true,
  "value": "Oh that's Andy. He's just hanging out.",

  "season": 8,
  "episode": 22,
  "deleted": false

}

Response 400

Body

{
  "success": false,
  "error": "Invalid Episode."
}

Response 400

Body
