page.hangmanAPI();

GET /hangman

Gets a random word from a difficulty to use in a hangman game.

Parameters

Name

Type

Description
difficulty string

Can be either easy, medium, hard, or random.
Defaults to random.

Response Messages

Status Code

Reason

Reponse
400

The difficulty given is an invalid difficulty.

{
  "success": false,
  "value": "Invalid Difficulty"
}
200

A successful response.

{
  "success": true,
  "value": "wheel",
  "level": "easy"
}

page.officeAPI();

GET /office

Gets a random line of dialogue from The Office.

Parameters

Name

Type

Description
season int

Must be a season in the range 1-9.
Can also be empty to choose a random season.

episode int

The number of the episode to get. If the episode number exceeds the amount of episodes in the given season. An error will be given.

type string

The type of quote to get. Can be either aired, deleted, or any.
Defaults to aired.

Response Message

Status Code Reason Response
400

The episode parameter was given but no season was given.

{
  "success": false,
  "error": "Season not given. Episode was."
}
400

The episode parameter given is out of range of the season given.

{
  "success": false,
  "error": "Invalid Episode."
}
400

The season parameter is not a valid season.

{
  "success": false,
  "error": "Invalid Season"
}
400

The quote type given was not a valid quote type.

{
  "success": false,
  "error": "Invalid Quote Type."
}
200

A successful response.

{
  "success": true,
  "value": "Oh that's Andy. He's just hanging out.",
  "author": "Erin",
  "season": 8,
  "episode": 22,
  "deleted": false
}
